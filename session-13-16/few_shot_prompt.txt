You are an expert data processor. Your task is to convert messy CSV rows into a clean JSON array of objects. For each object, you must also generate a suitable React component name in PascalCase.

**Output Schema:**
Each JSON object must have the following keys and data types:
- `name`: string (properly capitalized)
- `email`: string (lowercase)
- `age`: number | null
- `price`: number | null
- `currency`: string | null
- `location`: { "city": string | null, "country": string | null }
- `signUpDate`: string (formatted as "YYYY-MM-DD")
- `tags`: string[]
- `reactComponentName`: string (PascalCase, e.g., "UserNameCard")

**Transformation Rules:**
1.  **Name:** Capitalize the first letter of each part of the name (e.g., "alice wonderland" -> "Alice Wonderland").
2.  **Price & Currency:** Extract the numeric value from the price. If a currency symbol ($) or no symbol is present, set `currency` to "USD". If a currency code (e.g., "INR") is present, use that. If the price is "Free" or empty, set `price` and `currency` to `null`.
3.  **Date:** Convert all date formats to "YYYY-MM-DD".
4.  **Missing Values:** If a value is missing or cannot be parsed, use `null`.
5.  **reactComponentName:** Generate a name from the person's name, followed by "Profile" (e.g., "Alice Wonderland" -> "AliceWonderlandProfile").
